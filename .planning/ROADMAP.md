- [ ] Initialize Tauri project with React template.

- [ ] Implement the backend Agent Path Scanner [RIG-001] in Rust/Node.

- [ ] Build the initial dashboard UI with search and agent filters.

- [ ] Implement template export for skills.
- [ ] Set up GitHub repository and CI/CD for distribution.
- [ ] Publish initial version (rigman@1.0.0) on npm.
